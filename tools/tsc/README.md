# Midjourney 人工智能程序的提示生成器

## 项目目标
开发一个网站，帮助用户生成 Midjourney 人工智能程序的提示。用户输入关键词后，网站将输出详细和有创意的描述，以激发人工智能生成独特和有趣的图像提示词。

## 功能需求
1. 用户输入关键词。
2. 调用 Deepseek 接口生成Midjourney 人工智能程序的提示词信息。
3. 输出详细和有创意的描述。
4. 生成的提示词信息可以导出为 Word 文档。

## 技术实现
- 使用 HTML5 和 CSS3 进行前端开发。
- 使用 JavaScript 调用 Deepseek API。
- 使用第三方库实现 Word 文档导出功能。

## 页面结构
- **主页**：用户输入关键词的界面。
- **结果页**：显示生成的提示词和导出选项。

## 样式设计
- 采用苹果风格的简洁设计。
- 使用响应式设计，确保在不同设备上良好显示。
